    Arno Gourdol

    Welcome to ui-js!

    I’m Arno and for many years I've been passionate about beautiful user interfaces that help you stay in the flow while being a delight to use. At Apple I led the macOS UI team, helping bring the Aqua UI of macOS X to the world. At Adobe I worked on crafting great experiences for creative profesionals, on the desktop, on mobile and on the web.

    ui-js is a set of polished and opinionated web components to build interactive applications with the web platform.

    I can’t wait to see what you will do with it. Enjoy!

    —Arno
