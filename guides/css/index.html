    ui-js CSS

    To style some standard HTML markup, use the following CSS classes.

    Simple Button

    <button class="ui-button">Push Me</button>

    Sizes

    <div class="ui-buttons"> <button class="ui-button xs">Extra-Small</button> <button class="ui-button sm">Small</button> <button class="ui-button md">Medium</button> <button class="ui-button lg">Large</button> <button class="ui-button xl">Extra-Large</button> </div>

    Variants

    <div class="ui-buttons"> <button class="ui-button">Default</button> <button class="ui-button primary">Primary</button> <button class="ui-button secondary">Secondary</button> <button class="ui-button danger">Danger</button> </div> <div class="ui-buttons"> <button class="ui-button minimal">Minimal</button> <button class="ui-button minimal primary">Primary Minimal</button> <button class="ui-button minimal secondary">Secondary Minimal</button> <button class="ui-button minimal danger">Danger Minimal</button> </div>

    States

    <div class="ui-buttons"> <button class="ui-button">Enabled</button> <button class="ui-button" disabled>Disabled</button> </div>
